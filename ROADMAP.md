# BharatMonitor Roadmap

## v0.1 — MVP (Current)
- [x] MapLibre GL India choropleth map (all 36 states + UTs)
- [x] Live issue feed with category filters
- [x] AI analysis via Claude
- [x] Citizen report form with AI classification
- [x] Vercel Edge Functions (classify + RSS proxy)
- [x] Demo data for all states

## v0.2 — Real Data
- [ ] Live RSS ingestion from 12 Indian feeds
- [ ] CPGRAMS public data integration
- [ ] Auto-deduplication of similar issues
- [ ] Issue aging (auto-resolve after 30 days)

## v0.3 — Languages
- [ ] Hindi UI translation
- [ ] Tamil, Telugu, Bengali, Marathi support
- [ ] Voice report (speech-to-text)

## v0.4 — Citizen Features
- [ ] WhatsApp bot integration
- [ ] SMS reporting (1800 number)
- [ ] Photo upload with issue report
- [ ] GPS auto-detect location

## v0.5 — Analytics
- [ ] State-level trend charts (30d history)
- [ ] Resolution rate tracking per department
- [ ] Most reported issues by district
- [ ] Heatmap by district (not just state)

## v1.0 — Government API
- [ ] REST API for government departments
- [ ] Webhook notifications to departments
- [ ] SLA tracking dashboard
- [ ] Docker self-hosted image
- [ ] White-label for state government portals
